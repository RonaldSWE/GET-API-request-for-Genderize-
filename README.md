# Gender Classifier API (Genderize Integration)

A simple **Express** REST API that classifies a person's name by calling the **Genderize API** and returning a processed response.

This project was built for a backend task that requires:
- External API integration
- Input validation
- Proper error handling
- Data processing logic
- CORS support
- Deployment readiness

---
